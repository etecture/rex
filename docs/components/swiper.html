Swiper

import { Swiper } from "@etecture/rex";
<Swiper>
  <img
    width={4000}
    height={200}
    alt={"lorem picsum"}
    src={"https://picsum.photos/4000/200"}
  />
</Swiper>

This component is a convenience wrapper which uses the useSwiper hook under the hood. The behaviour and configuration is the same. Checkout useSwiper for more.

Example

import { Swiper } from "@etecture/rex";

const Example = () => {
  return (
    <Swiper>
      <img width={4000} height={200} alt={"lorem picsum"} src={"https://picsum.photos/4000/200"} />
    </Swiper>
  );
};

scrollbarProps

ScrollbarProps | undefined

viewportRef

RefObject<HTMLElement> | undefined

The reference to the element that should scroll

dragRef

RefObject<HTMLElement> | undefined

The reference to the element that should handle the drag events Will use the viewportRef by default

Default:viewportRef

velocityMultiplier

number | undefined

Controls the base velocity strength when swiping, higher number means faster scroll after releasing the mouse

Default:1

dampenerMultiplier

number | undefined

Controls how fast the velocity subsides with each frame, the smaller the number the faster the velocity decreases with each frame. 0.96 will reduce the velocity by 4% each frame, 0.50 would reduce it by 50% each frame

Default:0.97

enabled

boolean | undefined

If false the hook will not do anything

Default:true
