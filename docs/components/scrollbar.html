Scrollbar

Customizable scrollbar component with high layout compatibility

import { Scrollbar } from "@etecture/rex";

The Scrollbar is by default in an "auto" mode. It will try to take as much space as possible from its container. If its contents cannot fit the available space the scrollbars will be shown.

<Scrollbar>
  <div>{longContent}</div>
</Scrollbar>

You can also limit the max and min height of the scrollbar container.

<Scrollbar mah={600} mih={150}>
  <div>{longContent}</div>
</Scrollbar>

With this the container will not get taller than 600px and always keeps at least 150px.

If you use the Scrollbar in a flex layout set the mih prop to 0 so it will not be forced to take more space than the flex parent.

Styling

To every part of the component a classname can be added.

classNames?: {
  root?: string;
  viewport?: string;
  verticalTrack?: string;
  verticalThumb?: string;
  horizontalTrack?: string;
  horizontalThumb?: string;
};

Since the default styles of the Scrollbar are layered, you can easily override everything.

h

Height<string | number> | undefined

Fixed height of the scrollbar container

w

Width<string | number> | undefined

Fixed with of the scrollbar container

mah

MaxHeight<string | number> | undefined

Max height of the container, if the content is less than that the scrollbar won't be used

mih

MinHeight<string | number> | undefined

The min height of the container

maw

MaxWidth<string | number> | undefined

Max height of the container, if the content is less than that the scrollbar won't be used

miw

MinWidth<string | number> | undefined

The min height of the container

visibility

"always" | "hover" | "never" | undefined

Controls when the scrollbar tracks are shown. "always" will always show the scrollbar as long as the content is scrollable. "hover" will only show the tracks when hovering over the container. "never" will never show tracks

Default:"hover"

horizontalScrollDisabled

boolean | undefined

Disables horizontal scrolling

Default:false

verticalScrollDisabled

boolean | undefined

Disables vertical scrolling

Default:false

viewportRef

MutableRefObject<HTMLDivElement | null> | undefined

An optional reference to the scroll container. This is the element on which the scroll events happen.

classNames

ScrollbarClassnames | undefined

Classnames for each part of the component
