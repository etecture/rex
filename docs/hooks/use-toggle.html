useToggle

import { useToggle } from "@etecture/rex";
const [isOpen, handler] = useToggle();

This hook stores a simple boolean state and exposes some handlers to quickly open, close or toggle the value.

// ...

<Modal opened={isOpen} />

// ...

<Button onClick={handler.toggle} />
<Button onClick={handler.open} />
<Button onClick={handler.close} />

The initial value can be set like this:

const [isOpen, handler] = useToggle({ initialValue: true });

boolean | undefined
